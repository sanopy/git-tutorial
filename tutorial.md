# Git入門



# `$ git clone`



# `$ git branch`

- 現在のブランチを確認

`$ git branch`

- 新たなブランチを作成

`$ git branch ブランチ名`



# `$ git checkout`

- ブランチの切り替え

`$ git checkout ブランチ名`

- 新たなブランチを作成し，ブランチを切り替え

`$ git checkout -b ブランチ名`



# `$ git add`

- `$ git add -p`

- `$ git add .`



# `$ git commit`

- `$ git commit -m 'コミットメッセージ'`



# `$ git push`

- `$ git push origin master`



# `$ git fetch`

- `$ git fetch origin`



# `$ git merge`



# `$ git pull`

- `$ git pull origin master`
